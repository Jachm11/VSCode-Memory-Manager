\doxysection*{Why use this?}

ansi-\/colors is {\itshape the fastest Node.\+js library for terminal styling}. A more performant drop-\/in replacement for chalk, with no dependencies.


\begin{DoxyItemize}
\item {\itshape Blazing fast} -\/ Fastest terminal styling library in node.\+js, 10-\/20x faster than chalk!
\item {\itshape Drop-\/in replacement} for \href{https://github.com/chalk/chalk}{\texttt{ chalk}}.
\item {\itshape No dependencies} (Chalk has 7 dependencies in its tree!)
\item {\itshape Safe} -\/ Does not modify the {\ttfamily String.\+prototype} like \href{https://github.com/Marak/colors.js}{\texttt{ colors}}.
\item Supports \href{\#nested-colors}{\texttt{ nested colors}}, {\bfseries{and does not have the \href{\#nested-styling-bug}{\texttt{ nested styling bug}} that is present in \href{https://github.com/jorgebucaran/colorette}{\texttt{ colorette}}, \href{https://github.com/chalk/chalk}{\texttt{ chalk}}, and \href{https://github.com/lukeed/kleur}{\texttt{ kleur}}}}.
\item Supports \href{\#chained-colors}{\texttt{ chained colors}}.
\item \href{\#toggle-color-support}{\texttt{ Toggle color support}} on or off.
\end{DoxyItemize}

\doxysubsection*{Nested styling bug}

{\ttfamily ansi-\/colors} does not have the nested styling bug found in \href{https://github.com/jorgebucaran/colorette}{\texttt{ colorette}}, \href{https://github.com/chalk/chalk}{\texttt{ chalk}}, and \href{https://github.com/lukeed/kleur}{\texttt{ kleur}}.


\begin{DoxyCode}{0}
\DoxyCodeLine{const \{ bold, red \} = require('ansi-\/styles');}
\DoxyCodeLine{console.log(bold(`foo \$\{red.dim('bar')\} baz`));}
\DoxyCodeLine{}
\DoxyCodeLine{const colorette = require('colorette');}

\doxysection*{Toggle color support}

Easily enable/disable colors.


\begin{DoxyCode}{0}
\DoxyCodeLine{const c = require('ansi-\/colors');}
\DoxyCodeLine{}
\DoxyCodeLine{// disable colors manually}
\DoxyCodeLine{c.enabled = false;}
\DoxyCodeLine{}
\DoxyCodeLine{// or use a library to automatically detect support}
\DoxyCodeLine{c.enabled = require('color-\/support').hasBasic;}
\DoxyCodeLine{}
\DoxyCodeLine{console.log(c.red('I will only be colored red if the terminal supports colors'));}
\end{DoxyCode}


\doxysection*{Strip A\+N\+SI codes}

Use the {\ttfamily .unstyle} method to strip A\+N\+SI codes from a string.


\begin{DoxyCode}{0}
\DoxyCodeLine{console.log(c.unstyle(c.blue.bold('foo bar baz')));}
\DoxyCodeLine{//=> 'foo bar baz'}
\end{DoxyCode}
