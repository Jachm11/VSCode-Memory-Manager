\doxysubsection*{circular}


\begin{DoxyCode}{0}

\end{DoxyCode}

\begin{DoxyCode}{0}
\DoxyCodeLine{var inspect = require('object-\/inspect');}

\doxysubsection*{var s = inspect(obj, opts=\{\})}

Return a string {\ttfamily s} with the string representation of {\ttfamily obj} up to a depth of {\ttfamily opts.\+depth}.

Additional options\+:
\begin{DoxyItemize}
\item {\ttfamily quote\+Style}\+: must be \char`\"{}single\char`\"{} or \char`\"{}double\char`\"{}, if present
\end{DoxyItemize}
