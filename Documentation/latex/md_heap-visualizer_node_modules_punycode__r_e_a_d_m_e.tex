Punycode.\+js is a robust Punycode converter that fully complies to \href{https://tools.ietf.org/html/rfc3492}{\texttt{ R\+FC 3492}} and \href{https://tools.ietf.org/html/rfc5891}{\texttt{ R\+FC 5891}}.

This Java\+Script library is the result of comparing, optimizing and documenting different open-\/source implementations of the Punycode algorithm\+:


\begin{DoxyItemize}
\item \href{https://tools.ietf.org/html/rfc3492\#appendix-C}{\texttt{ The C example code from R\+FC 3492}}
\item \href{http://opensource.apple.com/source/ICU/ICU-400.42/icuSources/common/punycode.c}{\texttt{ {\ttfamily punycode.\+c} by {\itshape Markus W. Scherer} (I\+BM)}}
\item \href{https://github.com/bnoordhuis/punycode/blob/master/punycode.c}{\texttt{ {\ttfamily punycode.\+c} by {\itshape Ben Noordhuis}}}
\item \href{http://stackoverflow.com/questions/183485/can-anyone-recommend-a-good-free-javascript-for-punycode-to-unicode-conversion/301287\#301287}{\texttt{ Java\+Script implementation by {\itshape some}}}
\item \href{https://github.com/joyent/node/blob/426298c8c1c0d5b5224ac3658c41e7c2a3fe9377/lib/punycode.js}{\texttt{ {\ttfamily punycode.\+js} by {\itshape Ben Noordhuis}}} (note\+: \href{https://github.com/joyent/node/issues/2072}{\texttt{ not fully compliant}})
\end{DoxyItemize}

This project was \href{https://github.com/joyent/node/blob/master/lib/punycode.js}{\texttt{ bundled}} with Node.\+js from \href{https://github.com/joyent/node/compare/975f1930b1...61e796decc}{\texttt{ v0.\+6.\+2+}} until \href{https://github.com/nodejs/node/pull/7941}{\texttt{ v7}} (soft-\/deprecated).

The current version supports recent versions of Node.\+js only. It provides a Common\+JS module and an E\+S6 module. For the old version that offers the same functionality with broader support, including Rhino, Ringo, Narwhal, and web browsers, see \href{https://github.com/bestiejs/punycode.js/releases/tag/v1.4.1}{\texttt{ v1.\+4.\+1}}.

\doxysubsection*{{\ttfamily punycode.\+decode(string)}}

Converts a Punycode string of A\+S\+C\+II symbols to a string of Unicode symbols.


\begin{DoxyCode}{0}
\DoxyCodeLine{// decode domain name parts}
\DoxyCodeLine{punycode.decode('maana-\/pta'); // 'mañana'}
\DoxyCodeLine{punycode.decode('-\/-\/dqo34k'); // '☃-\/⌘'}
\end{DoxyCode}


\doxysubsection*{{\ttfamily punycode.\+encode(string)}}

Converts a string of Unicode symbols to a Punycode string of A\+S\+C\+II symbols.


\begin{DoxyCode}{0}
\DoxyCodeLine{// encode domain name parts}
\DoxyCodeLine{punycode.encode('mañana'); // 'maana-\/pta'}
\DoxyCodeLine{punycode.encode('☃-\/⌘'); // '-\/-\/dqo34k'}
\end{DoxyCode}


\doxysubsection*{{\ttfamily punycode.\+to\+Unicode(input)}}

Converts a Punycode string representing a domain name or an email address to Unicode. Only the Punycoded parts of the input will be converted, i.\+e. it doesn’t matter if you call it on a string that has already been converted to Unicode.


\begin{DoxyCode}{0}
\DoxyCodeLine{// decode domain names}
\DoxyCodeLine{punycode.toUnicode('xn-\/-\/maana-\/pta.com');}
\DoxyCodeLine{// → 'mañana.com'}
\DoxyCodeLine{punycode.toUnicode('xn-\/-\/-\/-\/dqo34k.com');}
\DoxyCodeLine{// → '☃-\/⌘.com'}
\DoxyCodeLine{}
\DoxyCodeLine{// decode email addresses}
\DoxyCodeLine{punycode.toUnicode('джумла@xn-\/-\/p-\/8sbkgc5ag7bhce.xn-\/-\/ba-\/lmcq');}
\DoxyCodeLine{// → 'джумла@джpумлатест.bрфa'}
\end{DoxyCode}


\doxysubsection*{{\ttfamily punycode.\+to\+A\+S\+C\+I\+I(input)}}

Converts a lowercased Unicode string representing a domain name or an email address to Punycode. Only the non-\/\+A\+S\+C\+II parts of the input will be converted, i.\+e. it doesn’t matter if you call it with a domain that’s already in A\+S\+C\+II.


\begin{DoxyCode}{0}
\DoxyCodeLine{// encode domain names}
\DoxyCodeLine{punycode.toASCII('mañana.com');}
\DoxyCodeLine{// → 'xn-\/-\/maana-\/pta.com'}
\DoxyCodeLine{punycode.toASCII('☃-\/⌘.com');}
\DoxyCodeLine{// → 'xn-\/-\/-\/-\/dqo34k.com'}
\DoxyCodeLine{}
\DoxyCodeLine{// encode email addresses}
\DoxyCodeLine{punycode.toASCII('джумла@джpумлатест.bрфa');}
\DoxyCodeLine{// → 'джумла@xn-\/-\/p-\/8sbkgc5ag7bhce.xn-\/-\/ba-\/lmcq'}
\end{DoxyCode}


\doxysubsection*{{\ttfamily punycode.\+ucs2}}

\doxysubsubsection*{{\ttfamily punycode.\+ucs2.\+decode(string)}}

Creates an array containing the numeric code point values of each Unicode symbol in the string. While \href{https://mathiasbynens.be/notes/javascript-encoding}{\texttt{ Java\+Script uses U\+C\+S-\/2 internally}}, this function will convert a pair of surrogate halves (each of which U\+C\+S-\/2 exposes as separate characters) into a single code point, matching U\+T\+F-\/16.


\begin{DoxyCode}{0}
\DoxyCodeLine{punycode.ucs2.decode('abc');}
\DoxyCodeLine{// → [0x61, 0x62, 0x63]}
\DoxyCodeLine{// surrogate pair for U+1D306 TETRAGRAM FOR CENTRE:}
\DoxyCodeLine{punycode.ucs2.decode('\(\backslash\)uD834\(\backslash\)uDF06');}
\DoxyCodeLine{// → [0x1D306]}
\end{DoxyCode}


\doxysubsubsection*{{\ttfamily punycode.\+ucs2.\+encode(code\+Points)}}

\DoxyCodeLine{punycode.ucs2.encode([0x1D306]);}
