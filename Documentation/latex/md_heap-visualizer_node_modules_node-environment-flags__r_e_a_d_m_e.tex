\doxysection*{Notes}


\begin{DoxyItemize}
\item This module approximates what {\ttfamily process.\+allowed\+Node\+Environment\+Flags} provides in versions of Node.\+js prior to v10.\+10.\+0. Since {\ttfamily process.\+allowed\+Node\+Environment\+Flags} is based on \href{https://nodejs.org/api/cli.html\#cli_node_options_options}{\texttt{ {\ttfamily N\+O\+D\+E\+\_\+\+O\+P\+T\+I\+O\+NS}}} (introduced in v8.\+0.\+0), the set of supported flags for versions older than v8.\+0.\+0 is {\itshape highly theoretical}.
\item Version ranges are matched using \href{https://npm.im/semver}{\texttt{ semver}}.
\item This module is granular to the {\itshape minor} Node.\+js version number; {\itshape patch} version numbers are not considered.
\item Results for unmaintained (odd) versions of Node.\+js are based on data for the most recent L\+TS version; e.\+g., running this module against Node.\+js v7.\+10.\+0 will yield the same results as would v6.\+14.\+0.
\item Prior art\+: @ljharb\textquotesingle{}s \href{https://npm.im/util.promisify}{\texttt{ util.\+promisify}}
\end{DoxyItemize}
