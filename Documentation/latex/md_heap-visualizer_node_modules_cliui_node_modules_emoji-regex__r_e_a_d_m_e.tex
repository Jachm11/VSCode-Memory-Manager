{\itshape emoji-\/regex} offers a regular expression to match all emoji symbols (including textual representations of emoji) as per the Unicode Standard.

This repository contains a script that generates this regular expression based on \href{https://github.com/mathiasbynens/unicode-tr51}{\texttt{ the data from Unicode Technical Report \#51}}. Because of this, the regular expression can easily be updated whenever new emoji are added to the Unicode standard.
