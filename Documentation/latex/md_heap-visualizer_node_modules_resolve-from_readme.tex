\begin{quote}
Resolve the path of a module like \href{https://nodejs.org/api/globals.html\#globals_require_resolve}{\texttt{ {\ttfamily require.\+resolve()}}} but from a given path \end{quote}


\doxysection*{Install}


\begin{DoxyCode}{0}
\DoxyCodeLine{\$ npm install resolve-\/from}
\end{DoxyCode}


\doxysection*{Usage}

\`{}\`{}`js const resolve\+From = require(\textquotesingle{}resolve-\/from');

// There is a file at {\ttfamily ./foo/bar.js}

resolve\+From(\textquotesingle{}foo\textquotesingle{}, \textquotesingle{}./bar\textquotesingle{}); //=$>$ \textquotesingle{}/\+Users/sindresorhus/dev/test/foo/bar.js\textquotesingle{} \`{}\`{}\`{}

\doxysection*{A\+PI}

\doxysubsection*{resolve\+From(from\+Dir, module\+Id)}

Like {\ttfamily require()}, throws when the module can\textquotesingle{}t be found.

\doxysubsection*{resolve\+From.\+silent(from\+Dir, module\+Id)}

Returns {\ttfamily null} instead of throwing when the module can\textquotesingle{}t be found.

\doxysubsubsection*{from\+Dir}

Type\+: {\ttfamily string}

Directory to resolve from.

\doxysubsubsection*{module\+Id}

Type\+: {\ttfamily string}

What you would use in {\ttfamily require()}.

\doxysection*{Tip}

Create a partial using a bound function if you want to resolve from the same {\ttfamily from\+Dir} multiple times\+:


\begin{DoxyCode}{0}
\DoxyCodeLine{const resolveFromFoo = resolveFrom.bind(null, 'foo');}
\DoxyCodeLine{}
\DoxyCodeLine{resolveFromFoo('./bar');}
\DoxyCodeLine{resolveFromFoo('./baz');}
\end{DoxyCode}
