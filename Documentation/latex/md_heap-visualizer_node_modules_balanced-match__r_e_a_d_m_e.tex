Match balanced string pairs, like {\ttfamily \{} and {\ttfamily \}} or {\ttfamily $<$b$>$} and {\ttfamily $<$/b$>$}. Supports regular expressions as well!

\href{http://travis-ci.org/juliangruber/balanced-match}{\texttt{ }} \href{https://www.npmjs.org/package/balanced-match}{\texttt{ }}

\href{https://ci.testling.com/juliangruber/balanced-match}{\texttt{ }}

\doxysection*{Example}

Get the first matching pair of braces\+:


\begin{DoxyCode}{0}
\DoxyCodeLine{var balanced = require('balanced-\/match');}
\DoxyCodeLine{}
\DoxyCodeLine{console.log(balanced('\{', '\}', 'pre\{in\{nested\}\}post'));}
\DoxyCodeLine{console.log(balanced('\{', '\}', 'pre\{first\}between\{second\}post'));}
\DoxyCodeLine{console.log(balanced(/\(\backslash\)s+\(\backslash\)\{\(\backslash\)s+/, /\(\backslash\)s+\(\backslash\)\}\(\backslash\)s+/, 'pre  \{   in\{nest\}   \}  post'));}
\end{DoxyCode}


The matches are\+:


\begin{DoxyCode}{0}
\DoxyCodeLine{\$ node example.js}
\DoxyCodeLine{\{ start: 3, end: 14, pre: 'pre', body: 'in\{nested\}', post: 'post' \}}
\DoxyCodeLine{\{ start: 3,}
\DoxyCodeLine{  end: 9,}
\DoxyCodeLine{  pre: 'pre',}
\DoxyCodeLine{  body: 'first',}
\DoxyCodeLine{  post: 'between\{second\}post' \}}
\DoxyCodeLine{\{ start: 3, end: 17, pre: 'pre', body: 'in\{nest\}', post: 'post' \}}
\end{DoxyCode}


\doxysection*{A\+PI}

\doxysubsection*{var m = balanced(a, b, str)}

For the first non-\/nested matching pair of {\ttfamily a} and {\ttfamily b} in {\ttfamily str}, return an object with those keys\+:


\begin{DoxyItemize}
\item {\bfseries{start}} the index of the first match of {\ttfamily a}
\item {\bfseries{end}} the index of the matching {\ttfamily b}
\item {\bfseries{pre}} the preamble, {\ttfamily a} and {\ttfamily b} not included
\item {\bfseries{body}} the match, {\ttfamily a} and {\ttfamily b} not included
\item {\bfseries{post}} the postscript, {\ttfamily a} and {\ttfamily b} not included
\end{DoxyItemize}

If there\textquotesingle{}s no match, {\ttfamily undefined} will be returned.

If the {\ttfamily str} contains more {\ttfamily a} than {\ttfamily b} / there are unmatched pairs, the first match that was closed will be used. For example, {\ttfamily \{\{a\}} will match `\mbox{[}'\{\textquotesingle{}, \textquotesingle{}a\textquotesingle{}, \textquotesingle{}\textquotesingle{}\mbox{]}{\ttfamily and}\{a\}\}{\ttfamily will match}\mbox{[}\textquotesingle{}\textquotesingle{}, \textquotesingle{}a\textquotesingle{}, \textquotesingle{}\}\textquotesingle{}\mbox{]}\`{}.

\doxysubsection*{var r = balanced.\+range(a, b, str)}

For the first non-\/nested matching pair of {\ttfamily a} and {\ttfamily b} in {\ttfamily str}, return an array with indexes\+: {\ttfamily \mbox{[} $<$a index$>$, $<$b index$>$ \mbox{]}}.

If there\textquotesingle{}s no match, {\ttfamily undefined} will be returned.

If the {\ttfamily str} contains more {\ttfamily a} than {\ttfamily b} / there are unmatched pairs, the first match that was closed will be used. For example, {\ttfamily \{\{a\}} will match {\ttfamily \mbox{[} 1, 3 \mbox{]}} and {\ttfamily \{a\}\}} will match {\ttfamily \mbox{[}0, 2\mbox{]}}.
