\doxysubsection{Descripción detallada}
small class for calculating \mbox{\hyperlink{class_m_d5}{M\+D5}} hashes of strings or byte arrays it is not meant to be fast or secure 

usage\+: 1) feed it blocks of uchars with update() 2) finalize() 3) get hexdigest() string or \mbox{\hyperlink{class_m_d5}{M\+D5(std\+::string)}}.hexdigest()
