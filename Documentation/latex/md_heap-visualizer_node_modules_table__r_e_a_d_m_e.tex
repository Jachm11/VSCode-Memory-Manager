Produces a string that represents array data in a text table.



\label{_table-features}%
 \doxysubsection*{Features}


\begin{DoxyItemize}
\item Works with strings containing \href{https://en.wikipedia.org/wiki/Halfwidth_and_fullwidth_forms}{\texttt{ fullwidth}} characters.
\item Works with strings containing \href{https://en.wikipedia.org/wiki/ANSI_escape_code}{\texttt{ A\+N\+SI escape codes}}.
\item Configurable border characters.
\item Configurable content alignment per column.
\item Configurable content padding per column.
\item Configurable column width.
\item Text wrapping.
\end{DoxyItemize}
