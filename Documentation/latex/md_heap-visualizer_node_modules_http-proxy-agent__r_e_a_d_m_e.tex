\doxysubsection*{An H\+T\+T\+P(s) proxy {\ttfamily http.\+Agent} implementation for H\+T\+TP}

\href{https://travis-ci.org/TooTallNate/node-http-proxy-agent}{\texttt{ }}

This module provides an {\ttfamily http.\+Agent} implementation that connects to a specified H\+T\+TP or H\+T\+T\+PS proxy server, and can be used with the built-\/in {\ttfamily http} module.

{\bfseries{Note\+:}} For H\+T\+TP proxy usage with the {\ttfamily https} module, check out \href{https://github.com/TooTallNate/node-https-proxy-agent}{\texttt{ {\ttfamily node-\/https-\/proxy-\/agent}}}.

\doxysection*{Installation }

Install with {\ttfamily npm}\+:


\begin{DoxyCode}{0}
\DoxyCodeLine{\$ npm install http-\/proxy-\/agent}
\end{DoxyCode}

\DoxyCodeLine{var url = require('url');}
\DoxyCodeLine{var http = require('http');}

\DoxyCodeLine{}
\DoxyCodeLine{// HTTP endpoint for the proxy to connect to}
\DoxyCodeLine{var endpoint = process.argv[2] || 'http://nodejs.org/api/';}

\DoxyCodeLine{var opts = url.parse(endpoint);}
