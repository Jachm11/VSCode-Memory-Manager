{\bfseries{Esprima}} (\href{http://esprima.org}{\texttt{ esprima.\+org}}, B\+SD license) is a high performance, standard-\/compliant \href{http://www.ecma-international.org/publications/standards/Ecma-262.htm}{\texttt{ E\+C\+M\+A\+Script}} parser written in E\+C\+M\+A\+Script (also popularly known as \href{https://en.wikipedia.org/wiki/JavaScript}{\texttt{ Java\+Script}}). Esprima is created and maintained by \href{https://twitter.com/ariyahidayat}{\texttt{ Ariya Hidayat}}, with the help of \href{https://github.com/jquery/esprima/contributors}{\texttt{ many contributors}}.

\doxysubsubsection*{Features}


\begin{DoxyItemize}
\item Full support for E\+C\+M\+A\+Script 2017 (\href{http://www.ecma-international.org/publications/standards/Ecma-262.htm}{\texttt{ E\+C\+M\+A-\/262 8th Edition}})
\item Sensible \href{https://github.com/estree/estree/blob/master/es5.md}{\texttt{ syntax tree format}} as standardized by \href{https://github.com/estree/estree}{\texttt{ E\+S\+Tree project}}
\item Experimental support for \href{https://facebook.github.io/jsx/}{\texttt{ J\+SX}}, a syntax extension for \href{https://facebook.github.io/react/}{\texttt{ React}}
\item Optional tracking of syntax node location (index-\/based and line-\/column)
\item \href{http://esprima.org/test/ci.html}{\texttt{ Heavily tested}} ($\sim$1500 \href{https://github.com/jquery/esprima/tree/master/test/fixtures}{\texttt{ unit tests}} with \href{https://codecov.io/github/jquery/esprima}{\texttt{ full code coverage}})
\end{DoxyItemize}
