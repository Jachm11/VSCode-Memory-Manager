\begin{quote}
Get the first fulfilled promise that satisfies the provided testing function \end{quote}


Think of it like an async version of \href{https://developer.mozilla.org/en/docs/Web/JavaScript/Reference/Global_Objects/Array/find}{\texttt{ {\ttfamily Array\#find}}}.

\doxysection*{Install}


\begin{DoxyCode}{0}
\DoxyCodeLine{\$ npm install p-\/locate}
\end{DoxyCode}


\doxysection*{Usage}

Here we find the first file that exists on disk, in array order.


\begin{DoxyCode}{0}
\DoxyCodeLine{const pathExists = require('path-\/exists');}
\DoxyCodeLine{const pLocate = require('p-\/locate');}
\DoxyCodeLine{}
\DoxyCodeLine{const files = [}
\DoxyCodeLine{    'unicorn.png',}
\DoxyCodeLine{    'rainbow.png', // Only this one actually exists on disk}
\DoxyCodeLine{    'pony.png'}
\DoxyCodeLine{];}
\DoxyCodeLine{}
\DoxyCodeLine{(async () => \{}
\DoxyCodeLine{    const foundPath = await pLocate(files, file => pathExists(file));}
\DoxyCodeLine{}
\DoxyCodeLine{    console.log(foundPath);}
\DoxyCodeLine{    //=> 'rainbow'}
\DoxyCodeLine{\})();}
\end{DoxyCode}


$\ast$\+The above is just an example. Use \href{https://github.com/sindresorhus/locate-path}{\texttt{ {\ttfamily locate-\/path}}} if you need this.$\ast$

\doxysection*{A\+PI}

\doxysubsection*{p\+Locate(input, tester, \mbox{[}options\mbox{]})}

Returns a {\ttfamily Promise} that is fulfilled when {\ttfamily tester} resolves to {\ttfamily true} or the iterable is done, or rejects if any of the promises reject. The fulfilled value is the current iterable value or {\ttfamily undefined} if {\ttfamily tester} never resolved to {\ttfamily true}.

\doxysubsubsection*{input}

Type\+: {\ttfamily Iterable$<$Promise$\vert$any$>$}

\doxysubsubsection*{tester(element)}

Type\+: {\ttfamily Function}

Expected to return a {\ttfamily Promise$<$boolean$>$} or boolean.

\doxysubsubsection*{options}

Type\+: {\ttfamily Object}

\doxyparagraph*{concurrency}

Type\+: {\ttfamily number}~\newline
 Default\+: {\ttfamily Infinity}~\newline
 Minimum\+: {\ttfamily 1}

Number of concurrently pending promises returned by {\ttfamily tester}.

\doxyparagraph*{preserve\+Order}

Type\+: {\ttfamily boolean}~\newline
 Default\+: {\ttfamily true}

Preserve {\ttfamily input} order when searching.

Disable this to improve performance if you don\textquotesingle{}t care about the order.
