\doxysubsection*{Turn a function into an \href{https://nodejs.org/api/http.html\#http_class_http_agent}{\texttt{ {\ttfamily http.\+Agent}}} instance}

\href{https://travis-ci.org/TooTallNate/node-agent-base}{\texttt{ }}

This module provides an {\ttfamily http.\+Agent} generator. That is, you pass it an async callback function, and it returns a new {\ttfamily http.\+Agent} instance that will invoke the given callback function when sending outbound H\+T\+TP requests.

\doxysection*{Example }

Here\textquotesingle{}s a minimal example that creates a new {\ttfamily net.\+Socket} connection to the server for every H\+T\+TP request (i.\+e. the equivalent of {\ttfamily agent\+: false} option)\+:


\begin{DoxyCode}{0}
\DoxyCodeLine{var net = require('net');}
\DoxyCodeLine{var tls = require('tls');}
\DoxyCodeLine{var url = require('url');}
\DoxyCodeLine{var http = require('http');}
\DoxyCodeLine{var agent = require('agent-\/base');}
\DoxyCodeLine{}
\DoxyCodeLine{var endpoint = 'http://nodejs.org/api/';}
\DoxyCodeLine{var parsed = url.parse(endpoint);}
\DoxyCodeLine{}
\DoxyCodeLine{// This is the important part!}
\DoxyCodeLine{parsed.agent = agent(function (req, opts) \{}
\DoxyCodeLine{  var socket;}
\DoxyCodeLine{  // `secureEndpoint` is true when using the https module}
\DoxyCodeLine{  if (opts.secureEndpoint) \{}
\DoxyCodeLine{    socket = tls.connect(opts);}
\DoxyCodeLine{  \} else \{}
\DoxyCodeLine{    socket = net.connect(opts);}
\DoxyCodeLine{  \}}
\DoxyCodeLine{  return socket;}
\DoxyCodeLine{\});}
\DoxyCodeLine{}
\DoxyCodeLine{// Everything else works just like normal...}
\DoxyCodeLine{http.get(parsed, function (res) \{}
\DoxyCodeLine{  console.log('"response" event!', res.headers);}
\DoxyCodeLine{  res.pipe(process.stdout);}
\DoxyCodeLine{\});}
\end{DoxyCode}

\doxysection*{A\+PI }

\doxysection*{Agent(\+Function callback\mbox{[}, Object options\mbox{]}) → \href{https://nodejs.org/api/http.html\#http_class_http_agent}{\texttt{ http.\+Agent}}}

Creates a base {\ttfamily http.\+Agent} that will execute the callback function {\ttfamily callback} for every H\+T\+TP request that it is used as the {\ttfamily agent} for. The callback function is responsible for creating a {\ttfamily stream.\+Duplex} instance of some kind that will be used as the underlying socket in the H\+T\+TP request.

The {\ttfamily options} object accepts the following properties\+:


\begin{DoxyItemize}
\item {\ttfamily timeout} -\/ Number -\/ Timeout for the {\ttfamily callback()} function in milliseconds. Defaults to Infinity (optional).
\end{DoxyItemize}

The callback function should have the following signature\+:

\doxysubsection*{callback(http.\+Client\+Request req, Object options, Function cb) → undefined}

The Client\+Request {\ttfamily req} can be accessed to read request headers and and the path, etc. The {\ttfamily options} object contains the options passed to the {\ttfamily http.\+request()}/{\ttfamily https.\+request()} function call, and is formatted to be directly passed to {\ttfamily net.\+connect()}/{\ttfamily tls.\+connect()}, or however else you want a Socket to be created. Pass the created socket to the callback function {\ttfamily cb} once created, and the H\+T\+TP request will continue to proceed.

If the {\ttfamily https} module is used to invoke the H\+T\+TP request, then the {\ttfamily secure\+Endpoint} property on {\ttfamily options} {\itshape will be set to {\ttfamily true}}.
