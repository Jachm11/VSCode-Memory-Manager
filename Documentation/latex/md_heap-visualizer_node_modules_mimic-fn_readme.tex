\begin{quote}
Make a function mimic another one \end{quote}


Useful when you wrap a function in another function and like to preserve the original name and other properties.

\doxysection*{Install}


\begin{DoxyCode}{0}

\doxysection*{A\+PI}

It will copy over the properties {\ttfamily name}, {\ttfamily length}, {\ttfamily display\+Name}, and any custom properties you may have set.

\doxysubsection*{mimic\+Fn(to, from)}

Modifies the {\ttfamily to} function and returns it.

\doxysubsubsection*{to}

Type\+: {\ttfamily Function}

Mimicking function.

\doxysubsubsection*{from}

Type\+: {\ttfamily Function}
