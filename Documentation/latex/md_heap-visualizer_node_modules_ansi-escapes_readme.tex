\doxysection*{A\+PI}

\doxysubsection*{cursor\+To(x, y?)}

Set the absolute position of the cursor. {\ttfamily x0} {\ttfamily y0} is the top left of the screen.

\doxysubsection*{cursor\+Move(x, y?)}

Set the position of the cursor relative to its current position.

\doxysubsection*{cursor\+Up(count)}

Move cursor up a specific amount of rows. Default is {\ttfamily 1}.

\doxysubsection*{cursor\+Down(count)}

Move cursor down a specific amount of rows. Default is {\ttfamily 1}.

\doxysubsection*{cursor\+Forward(count)}

Move cursor forward a specific amount of columns. Default is {\ttfamily 1}.

\doxysubsection*{cursor\+Backward(count)}

Move cursor backward a specific amount of columns. Default is {\ttfamily 1}.

\doxysubsection*{cursor\+Left}

Move cursor to the left side.

\doxysubsection*{cursor\+Save\+Position}

Save cursor position.

\doxysubsection*{cursor\+Restore\+Position}

Restore saved cursor position.

\doxysubsection*{cursor\+Get\+Position}

Get cursor position.

\doxysubsection*{cursor\+Next\+Line}

Move cursor to the next line.

\doxysubsection*{cursor\+Prev\+Line}

Move cursor to the previous line.

\doxysubsection*{cursor\+Hide}

Hide cursor.

\doxysubsection*{cursor\+Show}

Show cursor.

\doxysubsection*{erase\+Lines(count)}

Erase from the current cursor position up the specified amount of rows.

\doxysubsection*{erase\+End\+Line}

Erase from the current cursor position to the end of the current line.

\doxysubsection*{erase\+Start\+Line}

Erase from the current cursor position to the start of the current line.

\doxysubsection*{erase\+Line}

Erase the entire current line.

\doxysubsection*{erase\+Down}

Erase the screen from the current line down to the bottom of the screen.

\doxysubsection*{erase\+Up}

Erase the screen from the current line up to the top of the screen.

\doxysubsection*{erase\+Screen}

Erase the screen and move the cursor the top left position.

\doxysubsection*{scroll\+Up}

Scroll display up one line.

\doxysubsection*{scroll\+Down}

Scroll display down one line.

\doxysubsection*{clear\+Screen}

Clear the terminal screen. (Viewport)

\doxysubsection*{clear\+Terminal}

Clear the whole terminal, including scrollback buffer. (Not just the visible part of it)

\doxysubsection*{beep}

Output a beeping sound.

\doxysubsection*{link(text, url)}

Create a clickable link.

\href{https://gist.github.com/egmontkob/eb114294efbcd5adb1944c9f3cb5feda}{\texttt{ Supported terminals.}} Use \href{https://github.com/jamestalmage/supports-hyperlinks}{\texttt{ {\ttfamily supports-\/hyperlinks}}} to detect link support.

\doxysubsection*{image(file\+Path, options?)}

Display an image.

{\itshape Currently only supported on i\+Term2 $>$=3}

See \href{https://github.com/sindresorhus/term-img}{\texttt{ term-\/img}} for a higher-\/level module.

\doxysubsubsection*{input}

Type\+: {\ttfamily Buffer}

Buffer of an image. Usually read in with {\ttfamily fs.\+read\+File()}.

\doxysubsubsection*{options}

Type\+: {\ttfamily object}

\doxyparagraph*{width}

\doxyparagraph*{height}

Type\+: {\ttfamily string $\vert$ number}

The width and height are given as a number followed by a unit, or the word \char`\"{}auto\char`\"{}.


\begin{DoxyItemize}
\item {\ttfamily N}\+: N character cells.
\item {\ttfamily Npx}\+: N pixels.
\item {\ttfamily N\%}\+: N percent of the session\textquotesingle{}s width or height.
\item {\ttfamily auto}\+: The image\textquotesingle{}s inherent size will be used to determine an appropriate dimension.
\end{DoxyItemize}

\doxysubsection*{i\+Term.\+annotation(message, options?)}

Creates an escape code to display an \char`\"{}annotation\char`\"{} in i\+Term2.

See the \href{https://iterm2.com/documentation-escape-codes.html}{\texttt{ i\+Term Proprietary Escape Codes documentation}} for more information.

\doxysubsubsection*{message}

Type\+: {\ttfamily string}

The message to display within the annotation.

The {\ttfamily $\vert$} character is disallowed and will be stripped.

\doxysubsubsection*{options}

Type\+: {\ttfamily object}

\doxyparagraph*{length}

Type\+: {\ttfamily number}\textbackslash{} Default\+: The remainder of the line

\doxyparagraph*{x}

Starting X coordinate.

Must be used with {\ttfamily y} and {\ttfamily length}.

\doxyparagraph*{y}

Type\+: {\ttfamily number}\textbackslash{} Default\+: Cursor position

Starting Y coordinate.

Must be used with {\ttfamily x} and {\ttfamily length}.
